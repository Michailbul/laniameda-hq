### Key findings to remember:

**Algorithm (2026 — Grok-powered):**
- Transformer model reads tweet text semantically — topic clusters matter
- 5-15% test audience first, then expansion if velocity is good
- Time decay: 50% visibility loss every 6 hours → post timing is critical
- Premium accounts: 2-4x reach multiplier (baseline for serious growth)

**Engagement weights (ranked):**
- Reply + author reply back = 150x a like (highest signal possible)
- Reply = 27x
- Quote tweet = 25x
- Retweet = 20x
- Profile click = 12x
- Bookmark = 10x
- Like = 1x (baseline)

**What's dead:**
- External links in main tweet → near-zero distribution since March 2026
- Long threads (15+ tweets) → time decay kills before momentum builds
- Engagement farming phrases → actively suppressed by Grok detection
- 5+ hashtags → spam signal
- Generic filler → Grok reads and scores quality of text

**What wins:**
- Short punchy single tweet (70-140 chars) with strong hook
- Replies in first hour to own post (150x signal)
- Bookmarkable frameworks/lists/prompts
- Native video (10x boost)
- Specific numbers in hooks
