### Key learnings
- Chrome headless PDF ignores `height: 297mm + overflow: hidden` for print — use `break-after: page` only, let content flow naturally
- `!important` CSS overrides don't reliably beat inline styles in Chrome's print engine — must replace inline styles directly in HTML via Python
- WeasyPrint creates blank pages with `page-break-after: always` on fixed-height divs
- For dark page contrast: rgba(255,255,255,X) where X < 0.7 on bright orange = invisible. Must use dark text (#1A0800) on bright orange backgrounds instead of white
