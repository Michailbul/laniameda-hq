### Bugs Fixed

**1. Missing SkillMdPreview component**
- Latest commit imported `./SkillMdPreview` but file was missing from repo
- Built component using `marked` (transitively available), styled with `--md-*` CSS tokens
- Committed fix: `fix: add missing SkillMdPreview component + md preview styles`

**2. Agent filter only showing subset of skills (e.g. Crea showing 4 instead of 23)**
- Root cause: `.claude/skills` was hardcoded as "ground truth" — agent filter intersected workspace with `.claude` library
- Fix: when `activeAgentFilter` is active, bypass `.claude` ground truth entirely in `skillsLab.ts` `filtered()` fn
- Commit: `fix: agent filter bypasses .claude ground truth — shows all workspace skills for agent`

**3. Skill count mismatch — sidebar says 24, panel shows 15 (Lani agent)**
- Root cause: ecosystem tab filter (`.openclaw`, `.agents` etc.) was applied on top of agent filter
- Ecosystem tab intersected Lani's 24 workspace skills with `/root/.openclaw/skills/` (17 overlap) → 15 shown
- Fix: wrap both `ecosystemFilter` application blocks in `if (!activeAgentFilter)` in `skillsLab.ts`
- Commit: `fix: ecosystem tab filter skipped when agent filter active — all workspace skills shown`

**4. Markdown preview contrast (light theme)**
- CSS variables `--st-text` etc were never defined; fallback was dark-mode white (`#e2e8f0`) on light background
- Fix: scoped `--md-*` tokens per `[data-theme="light"]` and `[data-theme="dark"]`
- Light: `#1e293b` body, `#0f172a` headings, `#f1f5f9` code bg
- Commit: `fix: md preview contrast — proper light/dark theme tokens`

### Features Shipped

**5. Family group headers — collapsible + threshold**
- Family groups now only appear if prefix has >5 members (was >1)
- Headers are collapsible with spring chevron animation
- `collapsedFamilies` Set state in SkillsLabV2
- Commit: `feat: family groups require >5 members + collapsible headers`

**6. Remove studio-name prefix family groupings**
- `LANIAMEDA-*`, `PARALLEL-*`, `PRODUCT-*`, `CREA-*` etc were auto-generated from skill dir prefix
- Fix: added `FAMILY_NOISE_PREFIXES` Set in `skillsLab.ts` — studio/agent name prefixes never used as family keys
- Commit: `fix: remove studio/agent name prefixes from skill family grouping headers`

**7. Color-coded dept pills — Canvas sidebar + Inspector tree**
- Before: flat grey monospace badge
- After: pill-shaped, color-coded by department, dot indicator, spring hover microinteraction
- Applied to: `SkillPaletteRow.tsx`, `SkillPreview.tsx`, `InspectorTree.tsx` (Canvas + Canvas2)
- CSS: `data-dept` attribute tokens for light and dark theme
- Spring hover: `cubic-bezier(0.34,1.56,0.64,1)` scale+brightness
- Commit: `feat: color-coded dept pills with spring hover on canvas sidebar`
- Commit: `feat: color-coded dept badges in inspector tree with spring hover`

**8. WCAG AA contrast fix for light theme badges**
- Previous light colors were too low contrast (rgba .12 bg + mid-range text)
- Fix: 700-level text colors + 1px tinted border (Linear/Vercel pattern)
- Commit: `fix: WCAG AA contrast on light theme dept badges — 700-level text + border`
