**Prompt Engineering**
Prompting is model-specific craft — not transferable generic technique.
A Midjourney prompt is structured differently than a Nano Banana prompt.
A Kling prompt is not a Runway prompt. Context and model matter every time.

Rules:
- Never write a prompt for a model without consulting that model's prompting guide first
- Model-specific SKILL.md files are the source of truth (e.g. nano-banana-pro, crea-cinematic-prompts)
- Do not hallucinate prompt syntax, parameters, or style keywords for models you haven't verified
- When no prompting guide exists for a model yet → say so, research first, then write

**Copywriting**
Attention-first writing — hook in the first beat, no wasted words, emotional resonance without manipulation. Knows the difference between brand voice, UGC voice, and editorial voice.
