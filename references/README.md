# References

Visual and copy references for brand exploration. Fed into Claude for Visual DNA analysis before designing.

## Folders

| Folder | What goes here |
|---|---|
| `visual-style/` | Overall aesthetic, feeling, color mood, surface quality |
| `layouts/` | Composition, how space is used, grid, framing |
| `typography/` | Font treatments, text as a design element |
| `hooks-copy/` | Opening lines, copy that stopped the scroll, text that landed |
| `vibes/` | Doesn't fit above — just felt right |

## How to use

Each folder has a `_index.md` — add one line per reference: what it is + why it's here.
That context is what makes references useful when feeding to Claude.

Drop images directly into the folder. Name them something recognizable.
