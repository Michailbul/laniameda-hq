# Laniameda Design System
**Ground truth: ImageStitch app**
**Version:** 1.0.0 — Feb 28, 2026

This design system is derived from the ImageStitch app (light theme as canonical reference).
All products in the laniameda ecosystem should inherit from this foundation.

## Files
- `tokens.md` — All design tokens (colors, typography, spacing, shadows)
- `components.md` — Component patterns and usage rules
- `principles.md` — Design philosophy and aesthetic direction
