# 00 — Brand Overview & Visual Philosophy

## Brand Identity

**Laniameda** is a personal AI creatorship vault — a visual workspace where creative professionals curate prompts, reference images, and generated media. The brand identity reflects the intersection of artisan craft and machine intelligence.

### Brand Name Treatment

- **Wordmark**: `laniameda` in display serif (Instrument Serif), lowercase
- **Product suffix**: `Gallery` in italic, slightly reduced opacity (0.8)
- **Logo mark**: A small ink-colored diamond (rotated 45° square, 2px border-radius), placed to the left of the wordmark
- **Logo mark size**: 10×10px at standard, 7×7px at sidebar scale
- **Logo mark color**: `--ink` (`#201710`)

### Brand Voice

- **Tone**: Quiet confidence — authoritative but never loud
- **Register**: Studio professional — speaks to visually literate creators
- **Not**: SaaS corporate, playful/whimsical, enterprise formal

## Visual Atmosphere

### Mood Keywords

`Warm editorial` · `Studio console` · `Paper & ink` · `Volcanic warmth` · `Artisan digital` · `Cinematic gallery`

### Atmosphere Description

The visual language draws from **printed art catalogues** and **film photography studios**. Surfaces feel like sun-warmed archival paper. Text reads like ink on cotton stock. Accents glow like embers — coral, amber, and warm red — never neon, never cold.

The UI recedes to let content (images, prompts) be the hero. Chrome is minimal. Interactions are tactile but restrained — a subtle lift here, a warm glow there. Nothing bounces, nothing flashes.

### Emotional Qualities

| Quality | Expression |
|---------|-----------|
| **Warmth** | Paper-toned backgrounds, amber/coral accents, warm gradients |
| **Confidence** | Strong monospace labels, uppercase tracking, ink-dark text |
| **Craft** | Grain textures, editorial typography, careful spacing |
| **Focus** | Content-first layouts, dimmed unselected states, clean hierarchy |
| **Tactility** | Brutalist button offsets, subtle hover lifts, glass surfaces |

---

## Design Principles

1. **Content is hero** — Images and prompts dominate; UI chrome recedes into the background
2. **Warm editorial** — Paper backgrounds, ink text, serif display accents — never cold, never clinical
3. **Pillar-aware** — Accent colors shift dynamically with the active content category (pillar)
4. **Quiet confidence** — No loud hover effects, no neon glows, no bouncy animations
5. **CSS-first** — Prefer Tailwind `hover:`/`active:`/`focus:` utilities over JavaScript event handlers
6. **Progressive disclosure** — Essential information first, details revealed on demand
7. **Brutalist undertones** — Offset shadows, uppercase mono labels, sharp industrial elements mixed with warm surfaces

---

## Color Mode

**Light only.** No dark mode. The warm paper aesthetic is core to the brand and does not translate to dark surfaces.

The light theme uses warm off-whites (`#fffaf5`) rather than pure white, and near-blacks (`#201710`) rather than pure black. This creates a naturally warm, low-contrast feel that reduces eye strain without a dark mode.

## What This Is NOT

- ❌ Not a generic SaaS dashboard (no blue primary, no gray-100 backgrounds)
- ❌ Not a dark techy aesthetic (no glowing neons on dark surfaces)
- ❌ Not playful/whimsical (no rounded bubbly shapes, no emoji-heavy)
- ❌ Not enterprise formal (no dense data tables, no overwhelming chrome)
- ❌ Not Material/Bootstrap/generic (custom visual language throughout)
