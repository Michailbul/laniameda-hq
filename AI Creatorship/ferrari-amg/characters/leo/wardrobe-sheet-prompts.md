# Leo — Character Sheet Prompt Pack

> **6 character sheets, each a single composite render.** Every sheet is ONE Nano Banana Pro prompt that generates ONE image containing multiple views or assets arranged side-by-side against a consistent neutral studio background. Output is a reference sheet, not a single photograph.
> Always feed Leo's face character sheet (that Michael already generated) as the identity reference alongside each sheet prompt.
> Neutral grey or white seamless background across all panels. Soft natural studio daylight. Tack-sharp focus. **No text labels, no graphics, no brand logos, no watermarks, no captions inside the frame.**
> **Neckwear:** Leo wears a **black silk NECKTIE** (four-in-hand), NOT a bow tie. The knot is always loosened and pulled 4–5 inches below the open collar.

## Generation Notes

- **Always attach Leo's face character sheet** as identity reference alongside every sheet prompt. Nano Banana Pro will lean on the face ref for likeness; the text prompt holds everything else.
- **Composite sheets may need 2–3 rerolls.** Nano Banana Pro sometimes merges panels or drops one. If that happens, explicitly re-prompt: *"three clearly separated panels in a horizontal triptych, narrow gap between each panel, no panel merging."*
- **Body must read as ~90 kg, broader build.** If the render comes back model-lean or ripped, re-roll pushing *"mesomorph with a little softness, slight midsection fullness, broader shoulders, normal-guy body."*
- **Hair must read as fully unstyled.** If slick or product-set, re-roll with *"slept-on, natural cowlicks, messy bangs falling across forehead, never styled."*
- **Necktie, never bow tie.** If Nano Banana regresses to a bow tie, re-roll and emphasize *"a four-in-hand knot NECKTIE, a long tie hanging down the chest, absolutely NOT a bow tie."*
- **Suspender attachment** = button-attached with small black leather loop tabs. Not metal clips. Matte fabric, not shiny satin.
- **Sneakers are unbranded.** Any swoosh / stripes / logo = reject and re-roll.
- **Tattoo design lands on the first good Sheet 04 render.** Once a blackwork sleeve pattern lands clean, lock it visually and reuse as reference for every downstream Leo shot. Update `lock.md` Reference Images section with the winning render.
- **Sheet 06 (props) may also belong in the AMG car lock** — once generated, duplicate the prompt into `characters/mercedes-amg-gt3/lock.md` as the interior-dressing reference.
- Save winning renders to `characters/leo/refs/` and reference them in downstream shot prompts.
