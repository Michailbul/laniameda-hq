## MJ Run Strategy

1. **First pass:** run each prompt with `--s 250` and no character ref. Check composition, lighting, vibe.
2. **Identity pass:** add `--cref <img-1-url> --cw 100` to character shots (R1, R4, R5, G2). Higher `--cw` forces stronger face lock.
3. **Style pass:** if R-series results feel off-vibe, add `--sref <img-1-url> --sw 50` to transfer the red-wash mood.
4. **Variance:** use `--repeat 4` or generate 4x manually per prompt to get coverage.
5. **Upscale winners** with MJ's upscale or pass through Nano Banana for realism polish (see laniameda mj-nb2-pipeline skill).

| G4 | NB2 | Macro tactile + deep black paint, NB2 wins |
