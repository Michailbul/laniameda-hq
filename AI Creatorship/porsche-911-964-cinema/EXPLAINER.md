# Project Explainer

This project follows the AI Creatorship OS markdown contract.

- `project.md` holds the brief.
- `directions/*.md` hold creative exploration lanes.
- `scenes/*/scene.md` holds scene intent.
- `scenes/*/shots/*/shot.md` holds shot intent.
- `prompt-thread` files hold copyable prompt variations.
