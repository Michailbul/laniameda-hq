# Michael's Thought Vault

Raw, unfiltered thoughts dictated to Lani. No editing, no polish — just capture.

## Purpose
Every thought here is a seed. Downstream uses:
- **Marketing** — content angles, X threads, carousels, long-form
- **Positioning** — brand guidelines, ICP definition, voice/tone
- **Hiring** — what we look for, culture, values
- **Product** — feature ideas, product vision, user insight
- **Strategy** — where the studio is going, what bets to make

## Structure
```
thoughts/
  YYYY-MM-DD-short-slug.md   ← one file per thought/session
  README.md                  ← this file
```

## File format
Each thought file has:
- Date + tags
- Raw transcription (unedited)
- Lani's distillation (key ideas extracted)
- Downstream angles (what this could become)

## Rule
Nothing gets deleted. Raw thoughts are raw. Polish happens downstream.
