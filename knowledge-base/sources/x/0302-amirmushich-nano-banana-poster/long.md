## The Concept
"Modular typographic brand poster design" — a single AI prompt template that generates professional brand posters by swapping one variable (the brand).

- **Red Bull:** same grid structure, brand-accurate colors (red/blue/gold), F1 car top-down view

The workflow is explicitly shown: "Change one variable" → "Hit Generate, get unlimited assets."
