# Juice — 10 AI Font Presets for Brand Typography

## The Core Method

**Tools required:**
1. **Google Nano Banana** — visual generation
2. **LTX Elements** — font consistency & workflow organization (key differentiator)
3. **10 prompts below** — pre-built per font

**Why LTX Elements matter (critical insight):**
Without Elements: model defaults to Bold instead of Semibold, ignores kerning, hallucinates extra/missing text.
With Elements: adherence jumps +80–90%. Upload font as Element → tag it in prompt with `@[ELEMENT NAME]`.

**4-step workflow:**
1. Access Nano Banana (via LTX GenSpace)
2. Upload your font as an Element for consistency
3. Copy preset prompt → replace only `[INSERT YOUR TEXT HERE]`
4. Generate → consistent typography output

## How to Use Prompts

1. Copy prompt from LTX Project template
2. Replace `"INSERT YOUR TEXT HERE"` with your text
3. Change colors (type it in or use Color Picker [Alt+P] in LTX)
4. Tag the uploaded font Element with `@[ELEMENT NAME]`
5. Hit Generate

**Key rule:** Change only `[YOUR TEXT]` → get pro-level type instantly.
