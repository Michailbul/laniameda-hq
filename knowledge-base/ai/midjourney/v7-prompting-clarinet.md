# Midjourney v7 Prompting — Clarinet guide (distilled)

Source: Clarinet (Midjourney promptcraft educator), public Notion **Image Prompting Guide**.

## Mantra
**Want to see it? Say it unambiguously.**

V7’s prompt adherence is improved — missing details are often missing because they weren’t explicitly anchored.

## Capability notes
- ~**5 subjects** easily, up to **7** with careful prompting.
- **20+ individually specified details** possible with clear writing.

## Prompt-writing guidelines (v7)
- If it’s not controlled with the prompt, Midjourney makes it up.
- Chaotic tokens → you lose control of the canvas.
- Prompt should address all areas of the canvas: **style, subject, background**.
- Only articulate details not covered by archetypes and/or requiring anchoring.
- Instructions / conversation / commands belong in **Conversational Mode** only.
- Jargon / camera metadata / technical specs are weakly correlated → unreliable.
- Literary writing wastes processing time. Use dense visual description.
- Prompts are not captions — optimize for model clarity.

## Troubleshooting workflow
Fix one issue at a time. Don’t rewrite the whole prompt immediately.
Use `--draft` and/or `--relax` to save minutes.

### If v7 drops details: redundancy (“cowbell”)
Use repetition and explicit phrasing to force attention.

Example:
- Wanted: “a man crying into his hands” (hands stay in lap)
- Better: **“a man crying into his hands covering his face”**

### If prompt must be rewritten from scratch
1) Start with **main subject** (most important first)
2) Add **setting/background**
3) End with **style details**

Move ignored elements closer to the start.

## Examples
- Open-ended: “This guy and his dog look exactly alike.”
- Creative: “twilight wild garden, impasto painting, painting-knife brushstrokes, deep-teal, dark-crimson, muted colors, overcast atmosphere, garden details”
- Highly specific: (the long beach/ocean-purple-wine example)
