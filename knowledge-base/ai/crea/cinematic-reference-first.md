# Crea Knowledge — Cinematic Reference-Frame-First (Higgsfield Soul 2 / Nano Banana 2)

## Principle
**To get cinematic AI video, you start with cinematic AI images.**

Treat the still image as the *ingredient* / *reference frame* for video.
Spend the effort on concepting + storyboarding + shot selection first.

## Default Workflow
1) **Concept**: 1–2 sentences describing what happens.
2) **Storyboard**: 3–5 shots (establishing → medium → close-up → detail).
3) **Hero still**: generate 1–3 cinematic stills; pick the best.
4) **Video prompts**: generate 2–3 prompts derived from the hero still.

## Cinematic Still Checklist
A prompt is not “cinematic” unless it explicitly states:
- **Lighting plan** (motivated source, contrast)
- **Composition** (default: off-center / rule-of-thirds)
- **Time-of-day + grade** (sunset, dusk, neon night, overcast)
- **Depth** (foreground/mid/background separation; controlled blur)

### Composition defaults
- Prefer **left-third / off-center** subjects.
- Use centered symmetry only when intentionally requested.

## Master Video Prompt Formula
Write video prompts in this exact order:

**Camera → Subject → Action → Environment → Lighting → Texture → Audio**

### Template
- Camera: [shot type] + [movement] + [lens feel]
- Subject: [who + wardrobe + defining details]
- Action: [micro-action; natural; no exaggeration]
- Environment: [where + background motion]
- Lighting: [motivated source + contrast + time]
- Texture: [realism cues]
- Audio: [wind/room tone/ambience]

## Camera Move Menu
Use as selectable presets:
- Static, Handheld
- Zoom In/Out
- Camera follows
- Pan L/R, Tilt U/D
- Orbit around
- Dolly in/out

## Negative Constraints (defaults)
- Motion realism: **no exaggerated movements**
- Character dataset consistency: **avoid duplicates, group shots, filters, face coverings (masks/sunglasses)**

## Example Snippets (prompt patterns)
- "Slow steady zoom in… strong natural wind… no exaggerated movements… steam rising…"
- "Tight medium close-up… prop left third… text square to camera… brief beat… hand enters… lifts prop out of frame…"

## How Crea Should Respond (deliverables)
For any cinematic request, output:
1) **Hero still prompt**
2) **2–3 video prompts** (establishing + action + close-up)
3) **Camera move choice**
4) **Negatives**
