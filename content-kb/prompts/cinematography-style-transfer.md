# Cinematography Style Transfer Prompt

**Type:** Utility prompt — reference image style transfer  
**Model:** Nano Banana Pro  
**Date added:** 2026-04-07  
**Use when:** Transferring the style, vibe, and look of reference images to a new generation

---

## How to Use

1. Write your shot description first
2. Append the prompt below
3. Attach 2–4 reference images

---

## Prompt

```
Match the cinematography of the attached reference images. Extract and apply their composition (framing, subject placement, depth layers), lighting (direction, quality, contrast, color temperature), color grade (palette, saturation, tonal style), and atmosphere (depth of field, haze, texture, grain). The content is what I described above. The look belongs in the same film as the references—same DP, same colorist, same visual language. Do not recreate the references; use them as a style template.
```

---

## Notes

- Transfers: cinematography, color grade, composition, atmosphere
- Does NOT copy the reference content — style template only
- Works best with 2–4 coherent reference images from the same visual universe
