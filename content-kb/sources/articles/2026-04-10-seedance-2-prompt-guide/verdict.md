# Verdict: ImagineArt Seedance 2.0 Prompt Guide

**Verdict:** PARTIAL — useful reference library, not a complete prompting doctrine

**Why relevant:**
- Confirms a clean baseline structure for Seedance prompts: **Subject → Action → Camera → Style**
- Provides a broad prompt library across 14 categories, useful for pattern mining and category-specific ideation
- Adds practical shot vocabulary that maps well to commercial, narrative, POV, action, and ASMR video generation

**Why partial instead of definitive:**
- The article is stronger as a prompt example bank than as a deep model-behavior analysis
- Some examples are verbose or platform-specific rather than universally durable prompting rules
- It under-specifies when to use strict multi-shot structure vs compact single-shot prompts
- It does not go far enough on continuity logic, environment reaction physics, or ad-montage prompting patterns

**Bottom line:** Keep it in the studio KB as a reference source and prompt library. Use it to enrich Seedance category coverage, but rely on our evolving internal skill for stricter prompting logic and better production judgment.
