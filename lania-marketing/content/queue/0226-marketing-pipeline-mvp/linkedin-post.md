# LinkedIn Post Draft — The MVP content system for builders (no video)

I wasn’t “bad at content.”
I was just trying to *write* every time I wanted to post.

If you’re building products, learning fast, and still not posting — it’s not laziness.
It’s conversion debt.

Your work is happening.
But it’s not being converted into content-ready units.

Here’s the MVP system I’m using now (especially if video is out of scope):

**1) Capture (60 seconds/day)**
Don’t write posts. Capture raw material.
- a link
- a screenshot
- a voice note
- “today I shipped X / learned Y”

**2) Distill (turn raw into a reusable atom)**
Pick one:
- a rule (“If you see X, do Y”)
- a 3–5 step framework
- a checklist
- a before/after

**3) Draft (platform-specific)**
- X: 1 idea per post, concrete example
- LinkedIn: narrative + lesson + actionable
- Carousel: 7–9 slides, templated

**4) Approve (5 minutes)**
This is the only part that needs your taste.
Everything else can be delegated.

**5) Archive**
Save the final post + the raw capture.
That’s how your knowledge compounds.

If you only do one thing: start the daily 60-second capture habit.
It’s the smallest input that produces the biggest output.

What’s currently blocking your consistency — time, overwhelm, or not knowing what to say?
