# Workflow: Carousel Planning

Create a publishable carousel plan and slide copy from one topic.

## Required context

Load before planning:
1. `context/style/voice.md`
2. `context/style/writing-style-en.md`
3. `.claude/skills/Carousel/references/carousel-design-skill.md`

## Steps

1. Extract target audience and desired outcome.
2. Build an 8-10 slide narrative arc.
3. Define each slide:
   - objective
   - headline
   - body copy (short)
   - visual direction note
4. Validate flow and CTA.

## Output format

```markdown
# Carousel Plan: [topic]

## Audience
[who this is for]

## Core Promise
[what the carousel helps them achieve]

## Slides
### Slide 1
- Objective:
- Headline:
- Body:
- Visual note:

...
```

Save to `content/carousels/plans/MMDD-<slug>-YY.md`.
