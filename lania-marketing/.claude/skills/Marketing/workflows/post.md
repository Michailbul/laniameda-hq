# Workflow: Post Drafting

Create one primary high-quality post from topic or source material.

## Required context

Load before drafting:
1. `context/style/voice.md`
2. `context/style/writing-style-en.md`

## Inputs

- Topic or thesis (`$ARGUMENTS`)
- Optional source file (`@path`)

## Steps

1. Determine post type:
   - single post
   - thread
2. Draft structure:
   - hook
   - argument/mechanism
   - example/proof
   - CTA
3. Run quality gates:
   - specific and actionable
   - no empty hype
   - clear scannability
4. Produce final draft.

## Output

Save to `content/posts/MMDD-<slug>-YY.md`.

Use format:

```markdown
# Post: [topic]

## Final Draft
[post copy]

## Notes
- Post type:
- Audience:
- CTA intent:
```
