# Playwright

- **Category:** automation
- **Status:** ✅ Active
- **What it does:** Programmatic browser automation framework for reliable, scriptable browser control
- **What works:**
  - Cross-browser (Chromium, Firefox, WebKit)
  - Reliable scripted flows (login, scrape, screenshot)
  - Headless and headed modes
  - Great for deterministic tasks
- **What doesn't work / limitations:**
  - Requires script maintenance when UIs change
  - Not AI-driven — needs explicit selectors/logic
- **How we use it:** Scripted browser tasks, testing, structured data extraction; also underlies OpenClaw browser tool
- **Links:** https://playwright.dev
- **Last updated:** 2026-03-02
