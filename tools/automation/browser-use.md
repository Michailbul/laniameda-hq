# browser-use

- **Category:** automation
- **Status:** ✅ Active
- **What it does:** AI-driven browser automation — give it a goal, it browses the web to complete it
- **What works:**
  - Natural language task instructions
  - Form filling, navigation, data extraction
  - Works with LLM backends (OpenAI, Anthropic)
- **What doesn't work / limitations:**
  - Non-deterministic — may fail on complex sites with heavy JS
  - Slower than Playwright for scripted tasks
  - CAPTCHAs and auth walls
- **How we use it:** Agent-driven web tasks where scripted automation isn't practical
- **Links:** https://github.com/browser-use/browser-use
- **Last updated:** 2026-03-02
