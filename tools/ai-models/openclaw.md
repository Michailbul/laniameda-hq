# OpenClaw

- **Category:** ai-models
- **Status:** ✅ Active
- **What it does:** Agent runtime and orchestration platform — the environment Lani (AI assistant) runs in
- **What works:**
  - Multi-channel messaging (Telegram, Discord)
  - Sub-agent spawning and orchestration
  - Heartbeat scheduling
  - Browser control, file ops, shell exec
  - Node management (VPS + Mac)
  - Webhook hooks for agent notifications
- **What doesn't work / limitations:**
  - Self-hosted — requires maintenance
- **How we use it:** Primary runtime for Lani; all agent tasks, heartbeats, and integrations run through OpenClaw
- **Links:** Internal — `openclaw help`
- **Last updated:** 2026-03-02
