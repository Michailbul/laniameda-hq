# MidJourney

- **Category:** image-gen
- **Status:** 🧪 Testing
- **What it does:** Highly stylized AI image generation via Discord bot or web interface
- **What works:**
  - Exceptional aesthetic quality
  - Strong style consistency with `--sref`
  - Wide style range from photorealistic to abstract
- **What doesn't work / limitations:**
  - Requires Discord or web subscription
  - Less programmatic control vs API-based tools
  - Slower iteration loop than local/API models
- **How we use it:** Exploring for mood boards, cinematic frames, and reference image generation
