- **Category:** integrations
- **Status:** ✅ Active
- **What it does:** Integration layer that connects AI agents to 250+ external tools and APIs with managed auth
- **What works:**
  - Google Calendar integration (read/write events)
  - Managed OAuth — no token juggling
  - Works with LangChain, OpenAI, Anthropic tool calling
- **What doesn't work / limitations:**

  - Occasional latency vs direct API calls
- **How we use it:** Primary way agents access Google Calendar and other external services
