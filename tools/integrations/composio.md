# Composio

**Category:** integrations  
**Status:** ✅ Active  
**Link:** https://composio.dev/

## What It Does
Integration layer for AI agents — connects tools like Google Calendar, Gmail, and others via a unified API.

## What Works Well
- Google Calendar integration working via Composio
- Simplifies OAuth/API key management for multiple services

## Limitations / Gotchas
- `COMPOSIO_API_KEY` configured in `~/.openclaw/.env`
- Specific integrations vary — check Composio dashboard for active connections

## Env Requirements
`COMPOSIO_API_KEY` — set in `~/.openclaw/.env`
