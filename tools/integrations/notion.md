# Notion

- **Category:** integrations
- **Status:** ✅ Active
- **What it does:** Project management and knowledge base platform — our visual kanban and task tracker
- **What works:**
  - Task tracking with status, priority, assignee
  - Database views (kanban, table, calendar)
  - API for programmatic sync from agents
- **What doesn't work / limitations:**
  - API rate limits can slow bulk syncs
  - Relational DB structure requires careful schema design
- **How we use it:** Michael's live view of all studio work; agents sync tasks via Notion API after completing work
- **Links:** https://notion.so | https://developers.notion.com
- **Last updated:** 2026-03-02
